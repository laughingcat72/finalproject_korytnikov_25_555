ValutaTrade Hub
Консольное приложение для торговли валютами и криптовалютой.

Установка
Установите Poetry (если нет):

bash
curl -sSL https://install.python-poetry.org | python3 -
Клонируйте проект и установите зависимости:

bash
cd valutatrade-hub
poetry install
Для получения актуальных курсов создайте файл .env в корне проекта:

bash
EXCHANGERATE_API_KEY=ваш_ключ_от_exchangerate-api.com
Использование
Запустите приложение:
bash
poetry run python main.py
Основные команды (вводите после запуска):
Регистрация и вход:

register имя пароль - создать аккаунт

login имя пароль - войти в аккаунт

Торговля:

buy валюта количество - купить валюту (пример: buy BTC 0.5)

sell валюта количество - продать валюту (пример: sell EUR 100)

show-portfolio - посмотреть портфель

Курсы валют:

update-rates - обновить курсы из интернета

get-rate из в - узнать курс (пример: get-rate BTC USD)

show-rates - показать все доступные курсы

show-rates --top 5 - топ-5 криптовалют

Прочие:

help - помощь по командам

exit - выход из программы

Примечания
Для работы без интернета используются базовые курсы

Все данные хранятся в папке data/ в JSON-файлах

Пароли защищены хешированием

Логи операций в папке logs/

Поддерживаемые валюты
USD, EUR, RUB, BTC, ETH, SOL, JPY, GBP, CAD, AUD, SCR
